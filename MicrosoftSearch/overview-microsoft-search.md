# <a name="overview-of-microsoft-search"></a>Обзор Поиска (Майкрософт)

Поиск (Майкрософт) — это единое средство поиска в приложениях для работы Microsoft 365 и экосистеме Майкрософт. Со временем доступность Поиска (Майкрософт) в приложениях Microsoft 365 расширяется.


Поиск (Майкрософт) помогает пользователям находить правильные ответы, людей и контент для выполнения задач в приложении, в котором они работают. Возможности Поиска (Майкрософт): 
- Пользователи получают результаты в соответствии с **контекстом** приложения, из которого выполняется поиск. Например, при поиске в Outlook они находят письма, а не сайты SharePoint. При поиске в SharePoint они находят сайты, страницы и файлы, а не письма.
- Независимо от используемого пользователями приложения Поиск (Майкрософт) является **личным**.  Поиск (Майкрософт) использует аналитические сведения из Microsoft Graph, чтобы отображать интересующие каждого пользователя результаты. Для каждого пользователя могут отображаться разные результаты, даже если они выполняют поиск одинаковых фраз. Пользователям видны только те результаты, к которым у них есть доступ. Поиск (Майкрософт) не изменяет разрешения.   
- Пользователям не нужно запоминать место хранения сведений. Например, пользователь работает в Word и хочет использовать сведения из презентации, которой другой сотрудник поделился из своего хранилища OneDrive. Нет необходимости переходить в OneDrive и искать эту презентацию, можно просто выполнить поиск в Word.  
- При использовании Bing пользователи могут получать результаты как из Интернета, так и из организации. 

Поле Поиска (Майкрософт) отображается для пользователей в строке заголовка:

![Снимки экранов, демонстрирующие окна приложений со строкой Поиска (Майкрософт) в заголовке](media/Headings_400.png)


Когда пользователь щелкает в поле поиска, служба поиска предлагает результаты с учетом предыдущих действий пользователя в Office 365 и с учетом популярного контента в организации. Примеры действий, учитываемых поиском: файлы, с которыми пользователи недавно работали, недавно использованные команды, а также люди, с которыми они совместно работали. Когда пользователи начинают вводить запрос в поле поиска, выполняется обновление рекомендуемых результатов. Пользователи могут открывать результаты прямо из поля поиска.

![Снимки экрана, демонстрирующие поле Поиска (Майкрософт) с запросом и рекомендуемыми результатами](media/search_box.png)




Если предложения в поле поиска не соответствуют нужному контенту, нажатие клавиши ВВОД открывает полный список результатов. Пользователи могут использовать метаданные, например время и автора последних изменений, место хранения элементов, а также предварительно просмотреть его, чтобы определить, нужный ли это элемент.

![Снимки экрана со страницей результатов Поиска (Майкрософт)](media/SERP_text_520.png)



## <a name="benefits"></a>Преимущества 
**Поиск по Microsoft 365 из любого поля Поиска (Майкрософт)**. Пользователи могут выполнять поиск из любого поля Поиска (Майкрософт) и быстро возвращаться к своей работе. Поиск (Майкрософт) объединяет результаты из источников данных в Office 365, включая SharePoint, OneDrive для бизнеса и Exchange, а также из Интернета.

**Простой поиск**. Поиск (Майкрософт) предлагает результаты непосредственно в поле поиска с учетом предыдущей активности пользователей в Office 365.

**Поиск общих файлов**. Поиск (Майкрософт) поддерживает расширенные запросы, упрощающие поиск общих файлов. Пользователи могут легко находить файлы, над которыми они работают совместно.

**Отображение релевантного контента**. Продвигайте сведения и ответы, необходимые вашим пользователям для выполнения задач, например политики, сведения о льготах, ресурсы, инструменты и т. д. Вы также можете выбирать целевые группы получателей, такие как новые сотрудники и удаленные работники.

**Поиск (Майкрософт) развивается**. Набор типов результатов, которые могут искать пользователи, а также интеллектуальные функции поля поиска со временем расширяются.

**Администрирование во всех приложениях**. Поиск (Майкрософт) включен по умолчанию, и любые выполняемые административные действия применяются к Поиску (Майкрософт) во всех приложениях.

## <a name="which-apps-currently-have-microsoft-search"></a>Какие приложения в настоящее время поддерживают Поиск (Майкрософт)? 
Со временем доступность Поиска (Майкрософт) в приложениях Microsoft 365 расширяется.
Поиск (Майкрософт) в настоящее время предлагают следующие приложения Office 365:
- SharePoint Online 
- OneDrive для бизнеса
- Outlook в Интернете
- Приложения Office в Windows

Кроме того, пользователям доступен Поиск (Майкрософт) в:
- Bing 
- Windows
- Office.com
- Начальные страницы для Word, Excel и PowerPoint Online

Пользователей также могут выполнять поиск в Bing из адресной строки Microsoft Edge.

## <a name="requirements"></a>Требования 
В вашей организации должен быть клиент Office 365 с одной из следующих подписок:
- Office 365 бизнес базовый и бизнес премиум
- Office 365 A1/A3/A5
- Office 365 для образования E1/E3
- Office 365 корпоративный E1/E3/E3 для разработчиков/E5 
- Office 365 F1 
- Microsoft 365 бизнес 
- Microsoft 365 A3/A5 
- Microsoft 365 F1/E3/E5 

Только пользователи с активной учетной записью могут использовать Поиск (Майкрософт), и они должны выполнить **вход**.

## <a name="what-can-users-find-with-microsoft-search"></a>Что можно найти с помощью Поиска (Майкрософт)? 
Служба Поиска (Майкрософт) выполняет поиск в контенте, хранящемся в Outlook, OneDrive и SharePoint Online, включая людей из глобального списка адресов и группы Office 365. Если вы используете облачный гибридный поиск для настройки гибридной среды SharePoint, пользователи Поиска (Майкрософт) также смогут получать результаты из локального контента SharePoint, включая любой внешний контент, подключенный к вашей среде SharePoint Server. [Дополнительные сведения о гибридных средах поиска](https://docs.microsoft.com/ru-RU/sharepoint/hybrid/learn-about-cloud-hybrid-search-for-sharepoint).

Когда пользователи выполняют поиск на [домашней странице SharePoint](https://microsoft.sharepoint.com/_layouts/15/sharepoint.aspx) или сайте office.com, служба Поиска (Майкрософт) выполняет поиск во всей организации и выводит все найденные результаты, связанные с организацией. Это называется глобальной областью поиска.

Когда пользователи запускают поиск в Bing, служба Поиска (Майкрософт) выполняет поиск как в организации, так и в Интернете. В списке результатов пользователи получают широкий диапазон наиболее релевантных результатов из своей организации. Если нужно просмотреть все результаты, связанные с организацией, универсальная область поиска доступна всего одним щелчком мыши.

Когда пользователи выполняют поиск в приложении, они получают результаты с учетом контекста выполняемой работы. Например, если поиск выполняется из Outlook, результаты содержат письма, если поиск выполняется в SharePoint, результаты содержат списки сайтов, новостей и контента, а при поиске в PowerPoint результаты содержат действия или фразы из текущей презентации.

Ниже приведен обзор типа результатов, возвращаемых для пользователей в каждом приложении:

**Outlook**: сообщения электронной почты, вложения и люди из организации.

**Приложения Office в Windows**: действия в приложении, люди в организации и Интернете, файлы, объяснения слов, совпадения для запроса в файле или содержимом справки, контент в Интернете. Можно найти файлы Word, Excel, PowerPoint, Visio и OneNote.

**OneDrive**: можно найти файлы того же типа, что и в SharePoint.

**SharePoint**: файлы, папки, люди в организации, организационные диаграммы, сайты, страницы сайтов, новости, списки и элементы списков. Если определено, ответы на часто задаваемые вопросы, закладки, ведущие к достоверной информации, расположения на карте и инструменты. Узнайте, какие типы файлов можно найти.

**Bing**: контент в Интернете, файлы, группы Office 365, люди, беседы Yammer и Teams, организационные диаграммы, сайты SharePoint. Если определено, ответы на часто задаваемые вопросы, закладки, ведущие к достоверной информации, расположения на карте и инструменты.  Можно найти файлы Word, Excel, PowerPoint, OneNote и PDF.

**Windows**: файлы и пользователи.

**Office.com и начальные страницы Word, Excel и PowerPoint Online**: приложения, файлы, папки, люди, организационные диаграммы, сайты SharePoint, страницы сайтов, списки и элементы списков. Если определено, ответы на часто задаваемые вопросы, закладки, ведущие к достоверной информации, расположения на карте и инструменты. Здесь можно найти файлы тех же типов, что и в SharePoint.

## <a name="tailoring-microsoft-search-to-your-organization"></a>Настройка Поиска (Майкрософт) для организации

В качестве администратора вы может упростить получение достоверных результатов, связанных с организацией, при выполнении пользователями поиска на домашней странице SharePoint, сайте office.com или Bing.

**Отображайте полезный контент**. Помогайте пользователям находить важные средства и ресурсы в организации, создавая для них закладки. По аналогии с закладкой на общедоступной веб-странице вы можете создать закладку для любой внутренней веб-страницы, которую могут искать пользователи. Вы также можете интегрировать приложение Power App в закладку, чтобы пользователи могли выполнять свои задачи непосредственно из закладки.

**Предлагайте ответы на часто задаваемые вопросы**. Предоставляйте лучшие ответы на наиболее часто задаваемые вопросы. Когда пользователь вводит распространенный вопрос в поле поиска, Поиск (Майкрософт) отображает в качестве результата ответ, а не просто ссылку на веб-страницу.

**Отображение полезных расположений**. Показывайте на карте результаты и адреса для зданий, офисов и других рабочих пространств вашей организации. Пользователи могут использовать карты для построения маршрутов, просмотра окружения и многого другого.

Управление Поиском (Майкрософт) для всех приложений, содержащих поле Поиска (Майкрософт), выполняется из Центра администрирования Поиска (Майкрософт). 


## <a name="how-does-microsoft-search-work"></a>Как работает Поиск (Майкрософт)?
Поиск (Майкрософт) автоматически находит контент в SharePoint Online, OneDrive для бизнеса и Exchange. Если вы настроили гибридную среду SharePoint Server и среду поиска SharePoint Online с помощью облачного гибридного поиска, Поиск (Майкрософт) также сможет находить контент из локальной среды SharePoint, включая содержимое из локальных соединителей.

Когда пользователь выполняет поиск, Поиск (Майкрософт) обрабатывает запрос и выделяет цель поиска из больших фраз, используя искусственный интеллект (ИИ) для изучения распространенных избыточных фраз, которые добавляются пользователями в запросы и не влияют на цель поиска. Например, когда пользователь выполняет поиск по запросу "дополнительные сведения о том, как изменить пароль", менее важные слова извлекаются из запроса, и вызов результата осуществляется с учетом релевантных слов, таких как "изменить пароль".  

Результаты поиска, разрешенные для просмотра пользователю, отображаются на странице результатов поиска. Поиск (Майкрософт) использует интеллектуальные алгоритмы ранжирования, чтобы упорядочить результаты на основе релевантности.

## <a name="microsoft-search-in-sharepoint"></a>Поиск (Майкрософт) в SharePoint 
Поиск (Майкрософт) — это современный интерфейс поиска в SharePoint Online. В SharePoint Online также доступен классический интерфейс поиска. Администратор поиска не может отключить или включить ни современный, ни классический интерфейс поиска. Они оба включены по умолчанию. Предоставляемый пользователям интерфейс поиска зависит от места выполнения поиска:

- Пользователям доступно поле Поиска (Майкрософт) на домашней странице SharePoint, центральных сайтах, информационных сайтах и современных сайтах групп.
- Пользователям предоставляется классическое поле поиска на сайтах публикации, классических сайтах групп и в центре поиска.

Оба интерфейса поиска находят результаты в контенте SharePoint и OneDrive для бизнеса, а также людей в Office 365. 

Если вы продолжаете использовать классический поиск в SharePoint, [узнайте, как не затронуть интерфейс Поиска (Майкрософт) в SharePoint](https://docs.microsoft.com/ru-RU/sharepoint/differences-classic-modern-search).  

## <a name="microsoft-search-in-bing"></a>Поиск (Майкрософт) в Bing 
Когда пользователи выполняют поиск в Bing, они получают результаты поиска в Интернете и в организации, если они вошли в Office 365.
Поскольку поисковые запросы, связанные с работой, могут быть конфиденциальными, в Поиске (Майкрософт) реализован набор мер управления безопасностью, определяющий способы обработки таких запросов частью Bing для поиска общедоступных результатов в Интернете.

Независимо от того, содержит ли запрос пользователя один или несколько результатов, связанных с работой, в возвращаемом ответе, выполняются следующие меры: 

**Ведение журнала**. Все журналы поисковых запросов, относящиеся к трафику Поиска (Майкрософт), обезличиваются и хранятся отдельно от общедоступного, не относящегося к Поиску (Майкрософт) трафика. Они хранятся в течение 18 месяцев, и доступ к ним разрешен только для целей отладки. Запросы в этих журналах не используются для моделирования или отладки общедоступных функций, таких как автозаполнение или связанные поисковые запросы для общедоступной веб-части. Ограничение доступа выполняется посредством различных механизмов безопасности, включая группы безопасности и другие уровни в системе разработки.

**История поиска**. При выполнении входа с помощью рабочей или учебной учетной записи история поиска пользователя не будет доступна на других компьютерах или устройствах.

**Реклама**. Корпоративные поисковые запросы никогда не передаются и не предлагаются рекламодателям.
Журналы поисковой рекламы, относящиеся к Поиску (Майкрософт), хранятся отдельно от общедоступного трафика.
Реклама никогда не направляется пользователю на основе его рабочего удостоверения или организации.
