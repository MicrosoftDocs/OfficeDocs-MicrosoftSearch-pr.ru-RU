## <a name="communicates-with-the-trusted-cloud-from-the-browser"></a><span data-ttu-id="09f03-123">Связь с надежной облачной средой из браузера</span><span class="sxs-lookup"><span data-stu-id="09f03-123">Communicates with the Trusted Cloud from the browser</span></span>

<span data-ttu-id="09f03-p106">Когда пользователь выполняет вход с помощью своей рабочей или учебной учетной записи, Bing скачивает необходимые библиотеки клиента в браузер, чтобы включить результаты Поиска (Майкрософт).
